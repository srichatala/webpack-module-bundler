# Webpack module bundler
========================
Webpack module bundler application

## Pre-Requisities
1. [Node.js](https://nodejs.org/en/)
2. [Visual Studio Code] (https://code.visualstudio.com/?utm_expid=101350005-  28.R1T8FshdTBWEfZjY0s7XKQ.0&utm_referrer=https%3A%2F%2Fcode.visualstudio.com%2F)

## Clone
To get the application to on your local, follow the steps:

1. clone repository
2. C:/>webpack-module-bundler>npm install

## Build and Run
Build and Run the application by using webpack and react-hot

1. Open two terminals one is to build and other one is to run
2. C:/>webpack-module-bundler>npm run build ( To build) npm run watch (To see changes from command line)
3. React-hot live server
4. C:/>webpack-module-bundler>npm start
